# 웹디자이너를 위한 UI 설계 가이드

프로젝트 진행 시 협업을 위해 웹디자이너 알고 요구해야 할 부분을 정리하였습니다. <br>
기획과 개념이 잡힌 UI 설계를 통해 커뮤니케이션을 원활하게 하기 위해 제작된 가이드 입니다.

## 디자인 가이드
-----------------------------------------------------------------------------------------
1. 프로젝트 진행 전 사전 조사
2. 웹 사이트 제작 전 설계
3. 웹 디자인 이미지 저장 및 전달
4. UI 설계를 통한 디자인 적용


## 그리드 시스템
-----------------------------------------------------------------------------------------
1. 그리드 시스템을 사용하는 목적
2. 그리드 시스템 종류
3. 그리드 시스템과 베이스라인
4. 그리드 시스템 설계/적용 사례
